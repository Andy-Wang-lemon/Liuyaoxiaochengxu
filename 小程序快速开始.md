## 🚀 立即开始

### 第一步：启动后端服务器

确保后端服务正在运行：

```bash
cd /Users/shitou/Desktop/六爻开发

# 如果还没启动，运行：
DASHSCOPE_API_KEY=sk-096cc7facdbe4433a3a4681a787852f2 \
QWEN_MODEL=qwen-turbo \
PORT=3003 \
node dist/server-api.js
```

服务器应该显示：
```
⚠ 微信支付未配置，将使用模拟支付模式
六爻排盘服务已启动: http://localhost:3003
```

### 第二步：打开微信开发者工具

1. **下载工具**（如果还没安装）
   - 访问：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
   - 下载对应系统版本

2. **导入项目**
   - 打开微信开发者工具
   - 选择"导入项目"
   - 目录选择：`/Users/shitou/Desktop/六爻开发/miniprogram`
   - AppID：选择"测试号"（或输入你的小程序AppID）
   - 点击"导入"

### 第三步：开启本地调试

**重要！** 必须开启才能访问localhost

1. 点击右上角"详情"按钮
2. 勾选"不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书"
3. 关闭详情面板

### 第四步：编译运行

1. 点击工具栏的"编译"按钮
2. 在模拟器中查看效果
3. 尝试使用摇卦功能

## 🎯 关键配置

### 当前配置（开发模式）

| 配置项 | 当前值 | 说明 |
|--------|--------|------|
| AppID | 测试号 | 需要换成真实AppID |
| 服务器地址 | http://localhost:3003 | 开发用 |
| 支付模式 | 模拟支付 | 测试用，不产生真实交易 |
| AI模型 | qwen-turbo | 低成本快速模型 |

### 需要修改的地方（上线前）

**1. 修改AppID**
文件：`miniprogram/project.config.json`
```json
{
  "appid": "你的真实小程序AppID"
}
```

**2. 修改服务器地址**
文件：`miniprogram/app.js`
```javascript
globalData: {
  serverUrl: 'https://你的域名.com' // 改为实际域名
}
```

文件：`miniprogram/utils/api.js`
```javascript
const BASE_URL = 'https://你的域名.com/api'; // 改为实际域名
```

## 🔧 常见问题

### Q1: 编译报错"找不到模块"
**A**: 
- 检查文件路径是否正确
- 确认所有文件都已创建
- 重新导入项目

### Q2: 请求失败，提示"不在合法域名列表中"
**A**: 
- 确认已勾选"不校验合法域名"
- 重启微信开发者工具
- 检查后端服务是否运行

### Q3: 支付功能无响应
**A**: 
- 检查后端日志
- 确认模拟支付模式已启用
- 查看控制台错误信息

### Q4: 真机预览无法访问
**A**: 
- 真机无法访问localhost
- 需要部署到公网服务器
- 或使用内网穿透工具
