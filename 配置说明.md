# Gemini API 配置说明

## 步骤 1：获取 Gemini API Key

1. 访问 Google AI Studio: https://makersuite.google.com/app/apikey
2. 登录您的 Google 账号
3. 点击 "Create API Key" 创建新的 API 密钥
4. 复制生成的 API Key（格式类似：`AIza...`）

## 步骤 2：配置环境变量

### 方法一：使用 .env 文件（推荐）

1. 在项目根目录创建 `.env` 文件：
   ```bash
   cd /Users/Clara/Desktop/六爻开发
   touch .env
   ```

2. 在 `.env` 文件中添加：
   ```
   GEMINI_API_KEY=你的API密钥
   ```
   例如：
   ```
   GEMINI_API_KEY=AIzaSyBxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   ```

3. 安装 dotenv 包（如果还没有）：
   ```bash
   npm install dotenv
   ```

4. 在 `src/server.ts` 文件开头添加：
   ```typescript
   import 'dotenv/config';
   ```

### 方法二：直接在启动命令中设置

```bash
GEMINI_API_KEY=你的API密钥 npm run web
```

或在 `package.json` 的 `web` 脚本中设置：
```json
"web": "tsx --env-file=.env src/server.ts"
```

## 步骤 3：重启服务器

配置完成后，重启服务器：
```bash
PORT=3003 npm run web
```

## 验证配置

1. 完成摇卦后，点击"解卦（Gemini 3）"按钮
2. 如果配置正确，会看到 AI 生成的解卦内容
3. 如果未配置或配置错误，会显示提示信息

## 注意事项

- API Key 是敏感信息，不要提交到代码仓库
- 确保 `.env` 文件已添加到 `.gitignore`
- 免费 API 有使用限制，请合理使用
