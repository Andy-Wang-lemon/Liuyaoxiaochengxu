# 六爻摇卦小程序 - 部署指南

## 🎯 概述

本文档详细说明如何将六爻摇卦小程序从开发环境部署到生产环境。

## 📋 前置准备

### 1. 小程序账号

#### 注册小程序
访问：https://mp.weixin.qq.com/wxopen/waregister?action=step1

- **个人主体**：适合个人开发者，功能有限制
- **企业主体**：需要营业执照，功能完整
- **个体工商户**：介于两者之间

#### 获取AppID
1. 登录[小程序管理后台](https://mp.weixin.qq.com/)
2. 进入"开发" → "开发管理" → "开发设置"
3. 复制"AppID(小程序ID)"

### 2. 服务器准备

#### 域名要求
- 必须已备案
- 必须支持HTTPS（SSL证书）
- 建议使用CDN加速

#### 服务器配置
- 部署后端服务到服务器
- 确保端口对外开放（如3003）
- 配置Nginx反向代理（推荐）

## 🔧 配置步骤

### 步骤1：修改小程序代码

#### 1.1 配置AppID
编辑 `miniprogram/project.config.json`：
```json
{
  "appid": "你的小程序AppID",
  "projectname": "六爻摇卦"
}
```

#### 1.2 修改服务器地址
编辑 `miniprogram/app.js`：
```javascript
App({
  globalData: {
    serverUrl: 'https://你的域名.com' // 改为实际域名
  }
})
```

编辑 `miniprogram/utils/api.js`：
```javascript
const BASE_URL = 'https://你的域名.com/api'; // 改为实际域名
```

### 步骤2：配置小程序后台

#### 2.1 配置服务器域名
1. 登录[小程序管理后台](https://mp.weixin.qq.com/)
2. 进入"开发" → "开发管理" → "开发设置"
3. 找到"服务器域名"
4. 配置以下域名：

**request合法域名**：
```
https://你的域名.com
```

**注意**：
- 域名必须以 `https://` 开头
- 不能使用IP地址
- 不能带端口号
- 每月只能修改5次

#### 2.2 业务域名配置（可选）
如果使用web-view，需要配置业务域名：
1. 下载校验文件
2. 放到服务器根目录
3. 添加域名

### 步骤3：部署后端服务

#### 3.1 服务器环境
```bash
# 安装Node.js（推荐v18+）
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# 安装PM2（进程管理）
sudo npm install -g pm2
```

#### 3.2 上传代码
```bash
# 方式1：使用git
cd /var/www
git clone 你的仓库地址 liuyao
cd liuyao

# 方式2：使用scp
scp -r /Users/shitou/Desktop/六爻开发 用户@服务器IP:/var/www/liuyao
```

#### 3.3 安装依赖
```bash
cd /var/www/liuyao
npm install
npm run build
npm run build:frontend
```

#### 3.4 配置环境变量
创建 `.env` 文件：
```env
# 千问API
DASHSCOPE_API_KEY=sk-096cc7facdbe4433a3a4681a787852f2
QWEN_MODEL=qwen-turbo

# 服务器配置
PORT=3003

# 微信支付（如需真实支付）
WECHAT_APPID=你的应用ID
WECHAT_MCHID=你的商户号
WECHAT_SERIAL_NO=证书序列号
WECHAT_APIV3_KEY=APIv3密钥
WECHAT_NOTIFY_URL=https://你的域名.com/api/payment-notify
WECHAT_PRIVATE_KEY_PATH=./certs/apiclient_key.pem
WECHAT_PUBLIC_KEY_PATH=./certs/apiclient_cert.pem

# 管理后台
ADMIN_TOKEN=你的强密码
```

#### 3.5 启动服务
```bash
# 使用PM2启动
pm2 start dist/server-api.js --name liuyao

# 设置开机自启
pm2 startup
pm2 save

# 查看日志
pm2 logs liuyao

# 查看状态
pm2 status
```

### 步骤4：配置Nginx（推荐）

#### 4.1 安装Nginx
```bash
sudo apt update
sudo apt install nginx
```

#### 4.2 配置反向代理
创建配置文件 `/etc/nginx/sites-available/liuyao`：
```nginx
server {
    listen 80;
    server_name 你的域名.com;
    return 301 https://$server_name$request_uri;
}

server {
    listen 443 ssl http2;
    server_name 你的域名.com;

    # SSL证书
    ssl_certificate /etc/nginx/ssl/你的域名.pem;
    ssl_certificate_key /etc/nginx/ssl/你的域名.key;

    # SSL配置
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers HIGH:!aNULL:!MD5;
    ssl_prefer_server_ciphers on;

    # API代理
    location /api {
        proxy_pass http://localhost:3003;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }

    # 前端静态文件
    location / {
        proxy_pass http://localhost:3003;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
    }

    # 管理后台
    location /admin {
        proxy_pass http://localhost:3003/admin;
    }
}
```

#### 4.3 启用配置
```bash
sudo ln -s /etc/nginx/sites-available/liuyao /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx
```

### 步骤5：申请SSL证书

#### 方式1：Let's Encrypt（免费）
```bash
# 安装certbot
sudo apt install certbot python3-certbot-nginx

# 自动配置
sudo certbot --nginx -d 你的域名.com

# 自动续期
sudo certbot renew --dry-run
```

#### 方式2：阿里云/腾讯云证书
1. 在云服务商申请免费SSL证书
2. 下载Nginx版本证书
3. 上传到服务器 `/etc/nginx/ssl/`

### 步骤6：配置微信支付（可选）

如果要启用真实支付：

#### 6.1 申请微信支付商户号
访问：https://pay.weixin.qq.com/

需要：
- 企业/个体工商户资质
- 银行账户
- 法人身份证

#### 6.2 下载证书
1. 登录微信支付商户平台
2. 下载API证书
3. 上传到服务器 `/var/www/liuyao/certs/`

#### 6.3 配置小程序支付
1. 在小程序后台绑定商户号
2. 配置支付密钥
3. 测试支付流程

### 步骤7：测试和上传

#### 7.1 本地测试
1. 打开微信开发者工具
2. 取消勾选"不校验合法域名"
3. 测试所有功能是否正常

#### 7.2 真机预览
1. 点击工具栏"预览"
2. 扫码在真机测试
3. 检查功能是否正常

#### 7.3 上传代码
1. 点击工具栏"上传"
2. 填写版本号（如：1.0.0）
3. 填写项目备注
4. 点击"上传"

### 步骤8：提交审核

#### 8.1 登录小程序后台
访问：https://mp.weixin.qq.com/

#### 8.2 提交审核
1. 进入"版本管理"
2. 找到开发版本
3. 点击"提交审核"
4. 填写审核信息：
   - 选择类目：生活服务 → 算命/星座
   - 上传功能截图
   - 填写功能说明
   - 选择是否涉及用户信息

#### 8.3 等待审核
- 通常1-3个工作日
- 可在后台查看审核进度
- 如被驳回，根据反馈修改后重新提交

### 步骤9：发布上线

#### 9.1 审核通过后
1. 进入"版本管理"
2. 点击"发布"
3. 用户即可使用

#### 9.2 版本灰度
也可以选择灰度发布：
- 设置灰度比例（如10%）
- 逐步放量到100%

## ⚠️ 常见问题

### 问题1：域名配置不生效
**解决**：
- 检查域名是否已备案
- 检查SSL证书是否有效
- 等待小程序后台配置生效（可能需要几分钟）

### 问题2：支付功能无法使用
**解决**：
- 检查商户号是否绑定到小程序
- 检查证书路径是否正确
- 查看后端日志排查错误

### 问题3：API请求失败
**解决**：
- 检查服务器防火墙
- 检查Nginx配置
- 查看PM2日志
