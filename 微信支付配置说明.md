# 微信支付配置说明

## 1. 申请微信支付商户号

1. 访问微信支付商户平台：https://pay.weixin.qq.com/
2. 注册并完成企业认证
3. 获取以下信息：
   - APPID（应用ID）
   - MCHID（商户号）
   - API v3密钥
   - 证书序列号

## 2. 下载API证书

1. 登录微信支付商户平台
2. 进入"账户中心" -> "API安全" -> "申请API证书"
3. 下载证书文件（apiclient_key.pem 和 apiclient_cert.pem）
4. 在项目根目录创建 `certs` 文件夹
5. 将证书文件放入 `certs` 文件夹

## 3. 配置环境变量

在项目根目录的 `.env` 文件中添加以下配置：

```env
# 微信支付配置
WECHAT_APPID=你的应用ID
WECHAT_MCHID=你的商户号
WECHAT_SERIAL_NO=你的证书序列号
WECHAT_APIV3_KEY=你的APIv3密钥
WECHAT_NOTIFY_URL=https://你的域名.com/api/payment-notify

# 微信支付证书路径
WECHAT_PRIVATE_KEY_PATH=./certs/apiclient_key.pem
WECHAT_PUBLIC_KEY_PATH=./certs/apiclient_cert.pem
```

## 4. 配置回调URL

1. 登录微信支付商户平台
2. 进入"产品中心" -> "开发配置"
3. 设置支付回调URL为：`https://你的域名.com/api/payment-notify`
4. 确保你的服务器可以被外网访问

## 5. 测试环境

- 开发阶段可以使用微信支付的沙箱环境
- 沙箱环境申请：https://pay.weixin.qq.com/wiki/doc/api/jsapi.php?chapter=23_1

## 6. 注意事项

- 证书文件请妥善保管，不要上传到代码仓库
- `.env` 文件已在 `.gitignore` 中，不会被提交
- 生产环境必须使用 HTTPS
- 回调URL必须是公网可访问的地址
